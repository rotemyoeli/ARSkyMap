# OrbitMark Autopilot Cycle
Read `DOCS/09_AUTOPILOT_RUNBOOK.md`. Select one READY package. Invoke planning, domain, testing, evidence, council and release skills in order. Never skip a gate or combine unrelated packages. Finish with a durable status and state update. If externally blocked, continue independent work or return `blocked_external` without bypassing authority.
