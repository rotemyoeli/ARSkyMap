# Accessibility Gate
Run automated axe and ARIA snapshots, keyboard/focus, text scale, contrast, reduced motion and semantic checks. Complete the manual VoiceOver/TalkBack/device checklist when available. Any inaccessible critical task is STOP. Automated tools do not certify conformance.
